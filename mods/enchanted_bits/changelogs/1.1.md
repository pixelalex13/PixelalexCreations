## Additions

**Enchanting**

- Chiseled Bookshelves with three or more books can now be used to power an Enchanting Table
[gray]*The three-book minimum was implemented to prevent the chiseled bookshelf from becoming a complete alternative to the standard bookshelf.*[/]

**Tags**

**Enchanting Table Passthrough**
- Added a new tag, "enchanting_table_passthrough", which controls which blocks can be placed between an Enchanting Table and bookshelves
- Currently, the tag includes all non-full-cube blocks

## Changes

**Frost Walker Enchantment**
- Frost Walker can now be used with all rideable entities, such as boats, minecarts, horses, and more.
[gray]*This change was made to make Frost Walker more versatile and encourage its use.*[/]
