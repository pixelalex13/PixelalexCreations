*World Changes*

- Added the Fruma Province, an endgame region spanning levels 105-120 with five regions (Auburn Forest, Industrial District, Mistwoods, Highlands, Aelumia Citadel)
- Added a 13-part storyline for the Fruma Province
- Added 8 side quests for the Fruma Province
- Added a new raid: The Wartorn Palace
- Added a new boss altar: Deserter's Refuge
- Added several new Caves, Secret Discoveries, and World Events across Fruma
- Improved terrain across several areas
- Reworked several caves
- Reworked the dialogue system, which also allows for dialogue portraits to be used for some characters

*Class and Ability Changes*

- Added Ultimate Abilities for each archetype
- Increased the available AP from 45 to 50
- Added an eighth page to the Ability Tree
- Abilities now have custom VFX
- Reworked the Riftwalker archetype
- Began a partial rework of the Ritualist archetype

*Lootrun Changes*

- Added Pink Beacons which give +1 Beacon Reroll
- Added 2 new lootruns in Fruma: The Fruma Foray (West) and The Fruma Foray (East)

*Item Changes*

- Reworked the item UI
- Added Item Ascension, which allows certain items to be upgraded
- Powders can now be directly applied from the inventory
- Added Tier VII Powders
- Added 10 new Mythic Weapons, alongside over a thousand other new items

*Misc. Changes*

- Revamped Horses into Mounts, which also now includes Wyverns and Adasaurs
- Replaced Powder Masters with Item Upgraders
- Incorporated Liquid Emerald Merchants into Emerald Merchants
- Unidentifed items will now be more relevant to the current build the player is using
- Added Professions materials and ingredients for level 105-120
